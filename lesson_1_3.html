Adding CSS Style and HTML Structure

CSS = Cascading Style Sheets. They use inheritance (parent->child relationships).

Descendant elements inherit from any higher-up (ancestor) element which encloses them.

Added a DOCTYPE to the sample app.html; added a link in the <head> to the style.css file.

CSS - first write selector. Use element such as <h1>, or a class - element {}; .class {}.

Rules have declarations & properties; properties have values.

CSS properties:

  • font-style: normal; italic; oblique
  • font-weight: normal; bold; or, relative to parent, lighter; bolder; or numeric values 100, 200, 300, 400, 500, 600, 700, 800, 900.
  • See documentation for huge list.

Ways to use CSS in an HTML file:

  • Use a <style> element in the <head>
  • Link to a separate .css file
  • Use <link> tag (void) with rel="stylesheet" and href="filename.css".
  • Write style inline in the HTML. Messy! Deprecated.

divs and spans:

divs are used to logically organize code; or to group items together so that the whole block can be styled together. spans are used inline to style bits of text.

ids and classes:

ids are used to uniquely identify elements. Too many will slow a browser down.
You can use anchor elements to link directly to ids. (See Stage 1 Webcast on divs, spans, ids, classes.)

classes style groups of elements together. classes are more efficient than ids.

The Box Model

  • Content
  • Padding
  • Border
  • Margin

Size of element == sum of border, padding, content elements

Sizes can be percentages or pixels.

display: flex; overrides divs spanning the entire width of a page; lets boxes appear next to each other.
